# Issue Tracker

## System

This repository uses GitHub Issues:

- Repository: `matgCodes/ThreadManager`
- URL: `https://github.com/matgCodes/ThreadManager`
- External pull requests are not a triage request surface.

Use GitHub Issues for planned work, incoming requests, decision tickets, and
tracked implementation. Review pull requests separately as code changes.

## Routine operations

Use the GitHub CLI or GitHub API to:

- create and update issues;
- apply labels;
- assign and claim work;
- record resolution comments;
- close completed or rejected issues;
- query open, unassigned, blocked, and completed work.

Never store GitHub credentials in this repository.

## Wayfinding operations

- A Wayfinder map is a GitHub issue labelled `wayfinder:map`.
- Decision tickets are linked to the map as GitHub sub-issues.
- Use GitHub's native dependency relationship for blocking when available.
- Claim a ticket by assigning it before beginning work.
- The frontier consists of open, unblocked, unassigned child issues.
- Record a ticket's answer in a resolution comment before closing it.
- Refer to maps and tickets by linked title rather than bare issue number.
